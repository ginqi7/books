# 归途
由于昨晚的大雪，整个城市披上了银妆，厚重的白雪吸收了这繁华都市的喧嚣。银慵懒地坐在开往机场的车上，看着窗外飞速倒退的风景，不着边际地回忆着自己曾经从相反方向飞驰而来地片段。

「我来了！America。」银吻了吻胸前挂着的戒指，带着梦想推开机场厚重的大门，正式踏上纽约的土地。

第一次到达这陌生的城市，银显得十分兴奋，一切对于他来说都是新鲜的。他走出机场，用很搞笑的姿势与机场的标志留了张合影。

那时飞驰的汽车，把银带向了一个他完全没能预知的未来，当时的他还在心中默念着：「我会回来的，小叶。我会以全新的姿态出现在你面前。」

「如果流逝的时光也能像这倒退的风景一样回到原来的地方，那我会再次选择一个怎样的人生呢？」银看着窗外来来回回闪现的广告片，想着没多大意义的问题。他也知道，自己没办法找到这个愚蠢问题的答案，可是他还是不禁会这样去想。只要有选择、有得失，就会去想有「如果」。银习惯性的将撑着脑袋的右手在胸前摸索着，只是什么都没有。他低头看了一眼，尴尬的笑了笑。

就像是被按下「后退」的录影带，不同的画面快速的在眼前闪过，直到一切回到原点。

两个小时的车程终于到了目的地。银走下车，伸伸胳膊、动动筋骨。机场硕大的招牌再一次显现在他的眼里。银拿出手机，对着招牌摆出拍照的姿势，却没有按下快门。「额，我什么时候开始总是怀念过去了？难道我真的老了？不是吧，我还是个文艺小少年呢。」他还保持着拍照的姿势，乱七八糟的想着。

「哟！山姆大叔，我先回家咯。」银笑着走进厚重的玻璃门，和五年前的兴奋相比更加平静。所有陌生的城市都会给人新鲜感，寂寞感。可是无论活在哪个城市，生活永远是最朴实的柴米油盐。当剔除了浮华，银学会了用更平静的方式去热爱他的生活。

飞机的轰鸣声带着银飞向了他的家乡。

银上了飞机才发现了自己带错了书，抽象的内容让他没看多少页就靠着座椅睡着了，书掉到了座位的旁边，扉页上写着：

失去的东西虽然找不回来了，但是，忘却的记忆却是可以想起来的。

珍惜回忆。

——小叶。
